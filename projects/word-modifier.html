Spelling Word-Modifier

Description

This is a porgram that I developed mainly to assist with spelling homework. Our spelling tic-tac-toe requires us to modify words such as alphabetize, create word-stacks, word-stairs, and phone number words. With this in mind, I used my knowledge of algorithms to modify these words for me.

Challenges

This project was more about the logical and data structure side of coding which I was not yet familiar with. This got me really thinking about how efficient I could make my code. I had trouble trying to create algorithms that would do what I wanted. For the phone number modifier, it was difficult to map each letter to a different number in a way that didn't force me to duplicate lots of code.
